jquery-selectOption
===================

cross platform toggle(show/hide) select>option element

The example code is on http://jsfiddle.net/433okcL5/

Simple API

```js
$('select').selectOption();
$('select').hideAllOption();
$('select').showAllOption();
$('select').hideOption(':gt(1)');
$('select').showOption(':gt(1)');
```
